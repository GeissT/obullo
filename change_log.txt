CHANGE LOG:

Version 1.0.1 Changes

- Deleted old session.php from libraries folder.
- Global_controller meta tags moved to view_base_layout.php
- Added $params_or_no_ins instantiate switch to loader::model(); function. User can
switch off Model instantiate using by boolean like loader::model('model_name', false);  (added to User Guide)
- Added $params_or_no_ins Instantiate switch for all Loader Library functions.  
- Removed "use_bind_value" constant from constants/db.php file.
- renamed form helper validation_errors() function as "form_validate_errors();"
- renamed captcha helper create_captcha() function as "captcha_create();"
- Added source_url() function to Config.php class.
- Added Profiler Class. Removed some reserved controller variables (_libs and _mods)  from obullo/ob class.
- config/parents.php extend rules changed. Rules turned into associative arrays.
- Obullo Mvc2 changes: updated obullo.php, Global Controller extend rule changed belonging to 
$GLOBAL['d'] variable (directory request). Added '*' rule , All controllers in a folder which have the '*' rule before 
defined in parents.php will extend to its own Global Controller.
- removed $path param from css(); function , we use css('css/filename.css') instead of css('filename', $path = 'css');
- removed path from js(); function , we use js('js/welcome.js'); instead of js('filename') , added content_set_folder('img') support
to content_set_folder() and img() functions.
- mvc2 pattern improvements: added App_controller class to /parents folder. A App Contoller Class will control your
Global Controller files from now on.
- changed loader::file('folder/file') as loader::file('folder/file'.EXT) .
- added current_url(), current_dir(), current_class(), current_method() functions to Url helper.
- loading model from another path bug fixed. added 'models' string and DS.
- added $config['subhelper_prefix'] = 'my_';  to config.php file. 
- added sess_get_flash( $prefix = '', $suffix = '') func. parameters to session drivers.
- added base_url(), site_url() functions to url helper file.,
- added set  auto_base_url() , base_url and source_url functions to config class, updated source_url() function.
- changed css(filename.css), js(filename.js), img(filename.ext) functions as css(css/filename.css), js(js/filename.js), img(images/filename.ext);
- added extra ='' (attributes) param to form_hidden(); function..
- added content_view('', (object)$data); support ..
- added script(), script_app(), script_base() functions to head tag helper.
- ini_set('display_errors', ..); removed from bootstrap and config.php
- content helper renamed as view.php and all "content_" function prefix changed as "view_"
- some impromevents for profiler class, added php and php5 libraries colourful track support.
- added session helper driver extend support, added to all helpers extend support
- added i_request() function to base input helper.
- session function all_userdata changed as sess_alldata()
- changed view helper view_set_folder() 'app_view' switch case string as 'view_app'
- removed css() function array support from head tag helper
- loaded_helper() to common.php file, changed bootstrap.php and ob_set_headers func, added loaded helpers track support to profiler class.
- add config load and lang load track support for profiler class
- renamed lang_item() function as lang() in lang.php, updated all libraries.
- added auto_source_url(false); function to config class.
- added profiler_set($type, key, msg) and profiler_get($key) functions to common.php file, updated all libraries, removed ssc:intance()->_profiler_ variables.
